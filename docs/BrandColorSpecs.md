# iamshift.dev — Official Color Specification
### Brand Color System v1.0

## 1. Brand Color — Forge Orange  
Hex: #FF4E00  
Used for:
- Logomark
- Section headers
- Right-side nav highlight
- High-intent interactive moments
- Global accent identity

Forge Orange is immutable: no variants in branding contexts.

---

## 2. Brand Tint — Warm Orange  
Hex: #FF9500  
Used only for:
- Hover/focus states
- Subtle glow transitions
- Interaction feedback

Never used as a primary or identity color.

---

## 3. Supporting Colors  
Deep Blue #0C5DA5 (functional UI, links, accents)  
Tech Green #00AC6B (semantic success states)

These do *not* carry brand meaning.

---

## 4. Neutral System  
White #FFFFFF  
Gray surfaces  
Black #000000  
Text Gray #1A1A1A  
Used to frame content while keeping the brand colors dominant.

---

## 5. Dark Mode  
The site is currently light-mode only, but the dark palette remains defined for future expansion.

---

## 6. Usage Hierarchy  
Primary → Forge Orange  
Secondary → Warm Orange  
Supporting → Blue & Green  
Environmental → Neutrals

---

## 7. Notes  
Color system is implemented fully across:
- Boot sequence  
- Navigation  
- Global logo/title system  
- Section theming
